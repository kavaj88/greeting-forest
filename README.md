# 如是愿 (WISHING YOU)
# 你发过愿吗？许下的那愿，后来怎么样了？你默默祝福过某个人吗？你是否要剖析下自己内心深处的情感，把他记下来，放到一个安全的地方

一个基于 React + Vite + Supabase 的祈福祝愿 移动端、PC web 应用，，支持用户登录注册、发布心愿、点赞互动。

## 功能特性

- 🔐 **用户认证**: 邮箱 + 密码登录注册，验证码验证邮箱所有权
- 🏮 **三种心愿类型**: 祈福、祝愿、吐槽
- 🔒 **隐私保护**: 可选择公开或保密心愿
- 💫 **发光边框**: 不同颜色边框区分心愿类型
- 📱 **响应式设计**: 支持桌面端和移动端
- ✨ **动画效果**: 流畅的过渡和交互动画

## 闺蜜结婚，微信上那句"新婚快乐"太轻了
她下个月结婚。你们是十年朋友，一起经历过高考失利、一起熬过分手的夜晚、一起从实习生熬到转正。你要在微信上给她发什么？"新婚快乐，百年好合"？八个字，十年交情，你觉得不够。但你要在微信里发小作文，又太长了，她未必看得下去。

去"如是愿"给她发一条祝愿。50 字写核心祝福，800 字写你们的故事和你的真心话。你可以署你的名字，链接发给她。这是一条正式的祝福，不是微信聊天记录里一条被淹没的消息。**她可以单独保存，以后翻看的时候还在。**

## 考研 / 考公 / 高考，你需要一个仪式感
写一条心愿："今年必上岸。" 然后关掉页面，去刷题。

这个动作看起来没什么用，但心理学上叫**心理锚点**——你写下的那一刻，大脑会把它当成一个正式的承诺。考试当天你再回来看看这条心愿，你会发现它还在。它帮你记住你为什么努力。

考完之后你再看这条心愿，过了，你会回来还愿。没过，你也可以写一条新的继续。

## 分手了，话没说出口
在这里写下来。不为了让他/她看到，是为了让你自己**把话说完**。写完之后你会发现，你真正想说的不是挽回，不是解释，而是"我希望你好"或者"我也尽力了"。把这些话说出来，做一个人生的完结。

## 给异地朋友一个惊喜

他在北京，你在广州。他过生日，你发个红包吧，太敷衍。打个电话吧，怕打扰他跟朋友聚会。写张贺卡吧，快递赶不上。

在"如是愿"给他发一条祝愿，署你的名字，把链接发过去。他点进去看到的就是你专门为他写的祝福。这不是一条淹没在聊天记录里的消息，而是一个**独立存在的、专门属于他的页面**。

## 搬家 / 换工作 / 毕业，你想记住这一刻

你要离开这个城市了。四年了，你想发点什么纪念一下。但朋友圈的九宫格 + 文艺句子你已经发过太多次了，没什么新鲜感。而且朋友圈的信息流太快了，三天之后就被新的内容淹没了。

写一条心愿："毕业了，感谢这四年的所有人。" 然后你可以随时回来翻看。它不会消失，不会被新的内容覆盖。**它是一个记录，不是信息流里的一条。**

## 你只是想知道，别人都在想什么

你有时候会好奇：其他人每天都在想什么？他们也在焦虑吗？也在为生活奔波吗？还是只有你一个人觉得撑不下去？

浏览别人的心愿。你会发现：

- 一个妈妈在为孩子升学焦虑
- 一个程序员在抱怨加班
- 一个人在祝前任幸福
- 一个人在说"今天就想好好吃顿饭"

你会发现所有人都差不多。**没有人是真正轻松的，但所有人都还在坚持。** 这个认知本身就能给你力量。

## 你想给未来的自己留句话

有时候你会想：一年后的我会是什么样子？三年后的我呢？你现在纠结的事情，到时候还会是问题吗？但你没有记录的习惯，手机备忘录里的东西自己都不想看。

写一条心愿，设成保密（只有你自己能看到）。比如："希望一年后的你已经不再为这件事发愁了。" 一年后你回来打开看看，你会发现这比任何日记本都管用——因为它记录的不仅仅是你说了什么，而是**你当时有多认真**。

## 团队 / 班级 / 小组，想一起留个纪念

班级毕业、项目结项、团队解散。大家想在微信群里一起说点什么，但消息刷得太快，有人说"祝大家好"，然后就被下一个话题淹没了。没有仪式感，最后就是散了就散了。

让每个人在"如是愿"发一条祝愿，署上自己的名字，把链接发到群里。每个人点进来都能看到所有人写了什么。**这些内容是永久保存的，不是微信群里划过去就找不到的消息。** 以后同学聚会、团队重聚，翻出来看看，都在。

## "如是愿"解决了什么？

它解决的是一个被所有主流社交平台忽略的问题：

> **我们有很多话，不适合发朋友圈，不值得发微博，没地方跟朋友说，但我们需要一个地方把它们说出来。**

每一个场景，本质上都是同一个需求的不同表现形式：

| 你遇到了什么 | 用什么方式 | 得到什么 |
|------------|----------|---------|
| 家人生病焦虑 | 发一条"祈福" | 有人回应你，你没那么慌了 |
| 朋友结婚想认真祝福 | 发一条"祝愿" | 一份独立保存的正式祝福 |
| 工作压力大 | 发一条"吐槽" | 发现"我也是"，不孤独了 |
| 考试前想让自己定下来 | 写下心愿当心理锚点 | 记住自己为什么努力 |
| 分手了话没说完 | 在这里写完整 | 把话说完就好受了 |
| 异地朋友过生日 | 发一条专属祝愿 | 一个独立的、可保存的礼物 |
| 毕业/搬家/换工作 | 记录这一刻 | 不被信息流淹没的记忆 |
| 想知道别人怎么活的 | 浏览别人的心愿 | "原来不止我这样" |
| 想给未来的自己留句话 | 保密心愿 | 比备忘录管用一百倍 |
| 团队想一起留纪念 | 每人发一条 | 永久保存的集体记忆 |

## "如是愿，如是成"

这个名字的意思是：你说出的愿，终会结果。

这不是迷信。这是一个很简单的道理：**你把愿望写下来，你就离实现它近了一步。** 因为写下来意味着你认真想过，认真想过意味着你知道自己要什么，知道要什么意味着你会往那个方向走。

而如果你不是写给自己看，是写给所有人看——那还有另一层意义：**有人回应你，有人跟你一样，有人在替你加油。**
